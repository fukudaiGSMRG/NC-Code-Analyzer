🆕 最新アップデート (Ver.5.4)

今回のバージョンアップで、旋盤加工（溝入れ等）の解析精度が劇的に向上しました。

🔧 モーダル座標追跡機能の実装

これまでは「その行に座標値の記載がない」場合、移動がないものとして扱われていました。

Ver.5.4からは、前の行の座標（モーダル情報）を記憶・追跡することで、G01 Z... のように片側の軸しか記述がないブロックでも、現在のX座標を含めた正しい切削範囲として認識・出力します。

これにより、溝入れサイクルや固定サイクル等の解析精度向上。

⏱️ ドウェル時間(G04)のログ出力対応

G04 X... や P... U... で指定されるドウェル（一時停止）時間を自動検出し、ログ画面に「秒数/ミリ秒数」として明示的に表示するようにしました。

🐛 解析ロジックの修正

コメント () 内に数値が含まれる場合に誤検知するバグを修正しました。

旋盤モード時、G04のX値を「座標」として誤認しないよう除外処理を強化しました。

## ✨ 主な機能 (Features)

現場の「ヒヤリハット」を撲滅し、確認作業を効率化するための機能を搭載しています。

* **⚡ 早送り/切削の完全分離解析**
    * G00（位置決め）とG01/02/03（加工）の移動範囲を別々に算出。
    * 「早送りで加工対象に近づきすぎていないか」「切削Z深さは正しいか」を一目で判別可能です。
* **📊 ブロック別詳細リスト**
    * プログラム内の注釈 `(COMMENT)` を自動検出し、工程単位でリスト化。
    * Excelのような見やすいテーブル表示（ストライプ表示対応）。
* **📏 範囲チェック (Limit Checker)**
    * 別ウィンドウで機械のリミット値や干渉ラインを入力し、プログラムがその範囲を超えていないか自動判定します。
* **📈 グローバル・ダッシュボード**
    * 画面上部にプログラム全体の「XYZ最大最小」「最高回転数」「最大送り」を常時表示。
* **💾 CSVエクスポート**
    * 解析結果を詳細なCSVレポートとして出力可能。品質管理の記録として活用できます。
* **🔒 完全オフライン・セキュア**
    * 外部通信を行いません。機密性の高い図面データやノウハウを含むプログラムも、社内ネットワークから持ち出すこなく解析できます。

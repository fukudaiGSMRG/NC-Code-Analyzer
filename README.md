# NC Code Analyzer Pro (NCプログラム検証ツール)

<img width="2560" height="1440" alt="使用イメージ" src="https://github.com/user-attachments/assets/a7687099-4f3c-40eb-82bd-e380d0e471a8" />


**インストール不要、即起動。**
完全オフラインで動作する、プロ仕様のNCプログラム（Gコード）解析・検証ツールです。

工作機械（マシニングセンタ・旋盤）のプログラムをドラッグ＆ドロップするだけで、早送り/切削送りの詳細な座標範囲、回転数、送り速度を瞬時に解析します。

### ⚠️ 起動時のお願い (Note on Startup)

本アプリはインストール不要で使えるように設計されていますが、仕組み上、**起動（ダブルクリック）から画面が表示されるまでに 5秒〜15秒程度 かかる場合があります。**

故障ではありませんので、ダブルクリックした後は画面が出るまでそのままでお待ちください。（連打すると複数起動してしまいます）

## 🚀 ダウンロード (Download)

Python環境は不要です。（Releases）から最新の `.exe` ファイルをダウンロードしてください。



1. NC._Ver5.exe　をダウンロード。リンク先→　https://github.com/fukudaiGSMRG/NC-Code-Analyzer/releases/tag/V5.0.0
2. 任意のフォルダに保存。
3. ダブルクリックで起動します。

## ✨ 主な機能 (Features)

現場の「ヒヤリハット」を撲滅し、確認作業を効率化するための機能を搭載しています。

* **⚡ 早送り/切削の完全分離解析**
    * G00（位置決め）とG01/02/03（加工）の移動範囲を別々に算出。
    * 「早送りでワークに近づきすぎていないか」「切削Z深さは正しいか」を一目で判別可能です。
* **📊 ブロック別詳細リスト**
    * プログラム内の注釈 `(COMMENT)` を自動検出し、工程単位でリスト化。
    * Excelのような見やすいテーブル表示（ストライプ表示対応）。
* **📏 範囲チェック (Limit Checker)**
    * 別ウィンドウで機械のリミット値や干渉ラインを入力し、プログラムがその範囲を超えていないか自動判定します。
* **📈 グローバル・ダッシュボード**
    * 画面上部にプログラム全体の「XYZ最大最小」「最高回転数」「最大送り」を常時表示。
* **💾 CSVエクスポート**
    * 解析結果を詳細なCSVレポートとして出力可能。品質管理の記録として活用できます。
* **🔒 完全オフライン・セキュア**
    * 外部通信を行いません。機密性の高い図面データやノウハウを含むプログラムも、社内ネットワークから持ち出すことなく解析できます。

## 🛠️ 対応制御装置 (Supported Controllers)

* **FANUC** (Vertical Lathe / Turning) - G50/G96チェック対応
* **OSP** (Okuma Mill)
* **TOSNUC** (Toshiba Machine Mill)
